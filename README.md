# Python_Diwali_Sales_Analysis
This project involves performing Exploratory Data Analysis (EDA) on a Diwali sales dataset to identify key purchasing patterns. The goal is to provide actionable insights to enhance customer experience and increase revenue.
Data Cleaning: Using pandas to clean the data, which includes checking for null values, handling missing data, deleting unnecessary columns, and changing data types.
Exploratory Data Analysis (EDA): Using matplotlib and seaborn to visualize data based on different parameters: Gender: Analyzing purchasing power between male and female customers.
Age: Breaking down sales by age groups and gender
A final summary of insights gathered from the analysis to help the company make data-driven decisions.
